# Helpthread

Open-source, serverless helpdesk — shared inbox, threaded email conversations, knowledge base — for teams who live on Vercel and Supabase. Inspired by FreeScout, which proved the self-hosted ownership model; aimed at the bar Help Scout set for ease of use.

**Status: Phase 0 — foundations.** No code yet. Read [CHARTER.md](CHARTER.md) — it is the project's constitution and precedes any implementation.
